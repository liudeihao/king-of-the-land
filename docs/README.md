# 陆地之王：文档

所有游戏设计、规格与实现向约定**统一放在本目录**，仓库根目录仅保留 `README.md` 入口。代码（如 `Assets/`、Unity 工程等）可与之并列，不混用。

---

## 目录结构


| 位置              | 内容                                                                     |
| --------------- | ---------------------------------------------------------------------- |
| `**docs/`**（本层） | **概念与分层稿**：`00_`–`08_` 编号文件 + `VERSION_LOG.md`（思想版本与决策记录）              |
| `**docs/实现/`**  | **生产 / 实现向稿**：核心循环、系统结构、技术选型、微内核与 Mod、版本路线图、原型构建步骤，以及未来 `数值表.xlsx` 等资产 |


**约定**：

- 文内以仓库根为基准的路径书写，如 `docs/00_项目宣言.md`、`docs/实现/技术选型.md`（与 Git / IDE 中一致）。  
- `00_项目宣言` 的「未决问题统一维护于 06」即指 `docs/06_开放问题列表.md`。

---
