# 05 体验目标层

## 顶部元信息

### 当前共识摘要

- 体验重心不是操作快感，而是长期注视与见证：从混沌到秩序、自组织系统之美、以及“它开始思考了”的瞬间冲击。
- 关键体验情绪包括：饶有兴致的观察、好奇与发现、等待与震撼、以及干预的伦理张力。
- **观察本身就是选择，而选择会带来痛苦**：玩家无法观察一切，干预有不可预测的后果，不干预也是一种选择。最痛的时刻不是系统制造的，而是玩家自己的注视与缺席所产生的。- 体验目标应覆盖从"无事发生的宁静"到"文明涌现的震撼"的完整情绪光谱。
- 每次游戏都是独一无二的历史：不可复制性本身就是体验价值。
### 定性（原则 / 假设）

- 原则（体验方向与情绪目标）。

---

## 已确认（原则）

### 1. 观察者体验

- 以“创世之后的观察者/创造者”视角，饶有兴致地观看一个世界会发生什么。
- 长时间等待与注视（但情绪基调不必是孤独或无力）。

### 1.1 微观观察（不改变世界）

- 允许玩家把注意力聚焦到某个个体或群体，以便理解其行为与处境。
- 允许玩家对观察对象进行标记/命名/追踪，作为观看与记忆的工具，而非对世界的施令。
- **观察对象不限于猿类**：玩家可以同样深入地观察一只狼的捕猎生活、一头象的家族迁徙、一只鹰的领地巡逻。每个动物个体都有自己的故事。

### 1.2 叙事性描述（作为观看输出，而非世界内事实）

- 允许用叙事句式记录演化过程中的关键片段（例如：“某个个体在某地发现某物”）。
- 叙事中出现的名称可由玩家设定，用于记忆与追踪；在世界内部尚未具备文字与命名系统前，这些名称不应被视为群体共享的世界内知识。

### 2. 震撼与情绪峰值

- 见证复杂系统自发形成秩序的震撼。
- 进化的缓慢与偶然。
- 文明/意识可能出现时的巨大情绪冲击。
- “第一次意识诞生的瞬间”作为体验高潮的抽象目标。

### 3. 伦理张力

- 选择是否干预的伦理张力。
- 干预伴随责任感（不被当作奖励）。

### 3.5 困境体验：“必须选择”的情绪质感

- 本游戏的情绪高峰不仅来自“看到什么”，也来自“被迫选择”：
  - **注意力的痛苦**：你正在跟踪一只猿的一生，但通知提示你另一个区域正在发生重大事件。你切过去了——而你回来时，那只猿已经死了。你永远不知道它最后的时光是怎样的。
  - **救抟的痛苦**：你可以干预，但你知道干预会改变环境，而你无法预测改变的全部后果。你是在“用未知的恶果赌已知的善意”。
  - **见证的痛苦**：你选择不干预。世界按照其自身逻辑运行。你帮不了它们，你只能看着。但正因为你在看着——这种毯灭就不是无声的，而是“被见证的悲剧”。
  - **回溯的痛苦**：你可以回到灾难之前。但回溯会抹去其他所有生命在这段时间里的发展。你是在“用这些生命的未来换取那些生命的过去”。
- 这些困境不是预设的“关卡”，而是世界运行的自然结果与玩家情感投入的碰撞。**游戏不制造困境，但允许困境自然发生。**

### 4. 情绪光谱与节奏

- 体验不是单一情绪的持续输出，而是一个光谱：
  - **宁静与陪伴**：漫长时间里，世界只是在安静地运转。日出日落、季节更替、种群平稳地生存。这段"什么都没发生"的时间本身就是体验的一部分——它让后续的变化拥有重量。
  - **微小的惊喜**：某个个体偶然捡起石头敲碎坚果；某只幼崽模仿母亲的新行为；两个从未接触的群体在河边第一次相遇。这些事件本身很小，但在长期观察的背景下显得意味深长。
  - **紧张与担忧**：干旱持续延长、捕食者开始逼近核心领地、疾病在密集群体中扩散。玩家知道自己无法直接帮助，只能"看着"。
  - **丧失与哀悼**：被命名的个体死亡、被追踪多代的族群灭绝、一项延续了数百年的工具传统因最后一个掌握者的死去而消失。这些丧失是自然的，但因为玩家的注视而成为"被见证的悲剧"。
  - **震撼与敬畏**：跨越漫长等待之后，突然意识到"它们在有意识地做这件事了"——不是系统提示告诉你，而是你自己从行为模式的变化中察觉到的。

### 5. 涌现时刻（Emergent Moments）

- 以下是可能产生强烈体验冲击的"涌现时刻"（它们不是预设事件，而是系统运行可能自然产生的结果）：
  - **第一次工具使用**：从偶然的石头敲击到有意识地寻找合适的石头——玩家可能需要反复观察才能确认这不是随机行为。
  - **第一次火的保留**：一个群体在雷击后没有逃离，而是围着火源停留，并学会了添加燃料。
  - **第一次跨群体接触**：两个各自发展了不同传统的群体第一次见面——是冲突、是回避、还是交换？
  - **第一次埋葬行为**：一个个体死后，其同伴没有立即离去，而是做出了某种重复性的行为。这意味着什么？系统不会告诉你。
  - **第一次象征标记**：在洞穴墙壁或石块上留下了有规律的痕迹——是无意义的抓挠，还是表达的萌芽？
  - **第一次"对话"**：两个个体之间的声音交换开始呈现出交替模式和情境对应性。
  - **第一次注视星空**：一个个体在夜晚长时间抬头望向天空——没有功利目的，没有生存需求。这是好奇心的外显，是"存在感"的无声宣言。
### 5.5 个体叙事深度

- 模拟精度的目标之一是：**单个个体的一生应能产生类似小说的叙事张力**。
  - 一只猿的一生：幼年失去母亲、被另一个群体的雌性收养、成年后戞到一块边缘领地、在旱灾中失去一只后代、晚年孤独地死在曾经发现火的山洞里。
  - 一只狼的一生：出生时是狼群中最弱的幼崽、通过一次关键捕猎赢得地位、带领狼群度过严冬、被年轻的挑战者取代后离群、最终死于猿人设置的陷阱旁。
- **族群的发展应能产生类似史诗的叙事张力**：
  - 一个猿类族群从小河边的三只个体开始，经历分裂、合并、迁徙、技术突破、内部冲突、与其他物种的战争与共存，最终可能走向文明萌芽或走向灭绝。
  - 一个狼群从一对配偶开始，经历三代壮大成为区域顶级捕食者，然后因猿人的火技术而被迫迁移、最终与猿人族群形成微妙的共存关系。
- 这种叙事深度不是预先编写的剧本，而是个体级模拟的**自然结果**。
### 6. 对比与平行叙事的体验价值

- 当世界中同时存在多个种群/物种时，玩家可以在它们之间切换观察，感受不同路径的差异：
  - 同一环境压力下的不同应对策略。
  - 一个群体在兴盛、另一个群体在衰亡——玩家无法拯救后者。
  - 两条完全独立的演化路径最终产生了惊人相似的结果（趋同演化的震撼）。
  - 两个曾经相同的群体因地理隔离而走向完全不同的方向（分化的震撼）。- 平行叙事不仅限于猿类种群：玩家可以将观察焦点放在**任何物种的个体或群体**上。一个狼群的兴衰、一群大象的季节性迁徙、草原上食草动物与捕食者的动态平衡——这些都是完整且合法的观察体验。- 如果使用回溯与分支功能，玩家还可以体验到"同一个起点的不同命运"：这是对"偶然性"最直接的体验。

### 7. 长尾体验与可重玩性

- 每次游戏的世界历史都是独一无二的：不同的随机种子、不同的初始条件产生不可复制的叙事。
- 玩家的"观察日志"成为这个世界的唯一历史记录——世界消失后，日志是它存在过的证据。
- 不同初始条件之间的对比实验本身构成元游戏：极端条件下的世界（全冰期、极少水源、只有一个猿类种群）会产生什么？
- 分享性：玩家可能希望与他人分享自己观察到的世界历史——"在我的世界里，一个河谷部落在第3000年发明了陶器"。

### 8. 无胜负、无评分

- 没有"赢"和"输"：世界可以繁荣、可以衰亡、可以停滞，所有结果都是合法的。
- 没有数值评分或等级评定：不对玩家的世界设定或观察行为做好坏判断。
- 体验的"价值"由玩家自己在注视过程中赋予——系统不替玩家决定什么是"有意义的"。

### 9. 通知与提示系统（可配置）

- 系统可以在关键事件发生时向玩家发送通知提示：
  - 文化传统的消失（如：某种工具制造技术因最后一个掌握者死亡而失传）。
  - 物种或种群的灭绝。
  - 新行为的首次出现。
  - 跨群体的首次接触。
  - 被追踪个体/群体的重大事件。
- 所有通知类型都应提供**独立开关**：玩家可以自行选择哪些事件需要被提醒、哪些宁愿自己发现。
- 通知的设计原则：
  - 信息性而非解读性：告知"发生了什么"，不告知"这意味着什么"。
  - 不打断观察流：通知应以非侵入式的方式呈现（如边栏记录、小图标闪烁），而非强制弹窗。
  - 玩家始终可以选择"全部关闭"以获得纯粹的自主发现体验。
